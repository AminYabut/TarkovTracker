# Tarkov Tracker

An SPA that help Tarkov player tracking their game progress.

## Feature

- Task progress tracking
- Hideout progress tracking
- Searching item prices on fleamarket

## Getting Started

### Dependencies

- React
- Bootstrap v5.2
- the rest described in the package.json

### Installing

```
npm i
```

### Executing program

```
npm run start
```
